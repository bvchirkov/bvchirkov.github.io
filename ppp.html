Пакеты прикладных программ в сфере безопасности

Практическая работа

Цель

Изучение инструментов для моделирования пожара в здании

Задачи:

  1. формирование умения работы с прикладным программным обеспечением в сфере безопасности;
  2. развитие навыков работать с документацией;
  3. развитие навыков самостоятельной практической работы.

Ход работы

Часть I. Подготовка цифровой модели здания
  1. Ввести здание в программе Inkscape. Для ввода здания необходимо использовать расширение для программы Inkscape -- cfast-inkex.
  2. Сохранить цифровую модель здания в формате *.in для работы в CFAST.
  3. Соединить этажи с помощью горизонтальных вентиляционных отверстий через программу CEdit (устанавливается вместе с CFAST).
Часть II. Создание сценария
  1. Через программу CEdit настроить сценарий пожара. Для размещения пожара выбрат помещение ближайшее к эвакуационному выходу. Следовательно, сценариев должно получиться по количеству эвакуационных выходов. Один сценарий -- один файл.
    2. Химическая формула: C3H6.6O2.8, Soot Yield: 0.006, Co Yield: 0.043, Heat of Combustion: 14000 кДж/кг. Скорость развития пожара: средняя (medium). Максимальная мощность: 1 МВт. Площадь пожара: 0.25 м2. Местоположение: в центре помещения.
    Высота от пола: 0.
  2. Установить детекторы температуры в ближайших 10-15 помещениях, с которыми связано помещение с пожаром. Датчики температуры установить в центре помещения на высоте 170 см от пола.
  3. Установить длительность моделирования 3 минуты.
Часть III. Моделирование

Выполнить моделирование всех сценариев. Для каждого сценария определить время блокирования каждого помещения с датчиком по температуре (представить в виде таблицы).

Отчет по работе

Отчет по работе сдается в письменнов виде в формате pdf. Для оформления отчета воспользуйтесь LibreOffice (относится к тем, кто работает на компьютерах Университета). Наименование архива с файлами должно выглядеть следующим образом: Фамилия1_Фамилия2_cfast_ink.zip (Фамилия2 вставляется, если работа выполнялась вдвоем).
Папка для загрузки отчетов: CFAST_Labs.

Отчет включает 3 раздела:

  1. Файл с таблицей времени блокирования помещений по сценариям.
  2. Файл здания, выполненный в Inkscape (*.svg).
  3. Файлы сцернариев CFSAT (*.in).
  4. Файл с замечаниями и предложениями по плагину (*.txt).
  5. Файл, содержащий Ваш email для связи (*.txt).
