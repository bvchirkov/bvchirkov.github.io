Программирование

Ответ оформить в форме отчета по каждой задаче в одном файле и отправить его в формате pdf или в telegra.ph и направить ссылку на отчет.

Задачи на экзамен

Написать программы на языке Python, реализующие:

  1. Пусть a, b, c -- переменные, которым присваиваются случайные целые числа от 0 до 100, а переменная m в конечном итоге должна будет содержать значение наибольшей переменной. Вывести на экран значение переменной m;
  2. Пусть g -- список. Заполните список g десятью случайными числами. Каждое число должно быть не меньше 0 и не больше 100.
  3. В списке g (из предыдущего пункта) найти: среднее арифметическое (вывести значение на экран), минимальное значение (вывести индекс элемента на экран), максимальное значение (вывести индекс элемента на экран) и сумму трех наименьших чисел (вывести значение суммы на экран);
  4. Индивидуальные задания:
    1. Вывести таблицу значений функции x^2 + 2x - 3 с шагом 0.5. Минимальное значение x -- -4, максимальное -- 4;
    2. Вывести таблицу значений функции x^2 + 9x - 5 с шагом 0.5. Минимальное значение x -- -11, максимальное -- 2;
    3. Вывести таблицу значений функции x^2 - 9x + 5 с шагом 0.5. Минимальное значение x -- -1, максимальное -- 10;
    4. Вывести таблицу значений функции x^2 + 15x - 7 с шагом 0.5. Минимальное значение x -- -18, максимальное -- 3;
    5. Вывести таблицу значений функции 3x^2 + 7x - 9 с шагом 0.5. Минимальное значение x -- -4, максимальное -- 2;
    6. Вывести таблицу значений функции 2x^2 + 4x - 7 с шагом 0.5. Минимальное значение x -- -4, максимальное -- 2;
