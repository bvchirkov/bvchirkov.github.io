Практическая работа № 3. Массивы. Знакомство

Написать программу, реализующую:

  1. вычисление суммы чисел, входящих в массив;
  2. вычисление среднего арифметическго значения элементов, входящих в массив;
  3. поиск максимального значения;
  4. поиск минимального значения.

Где почитать

  • Массивы в языке Си
  • Массивы
  • Одномерные массивы

Практическая работа № 4. Массивы. Сортировка
  1. Написать программу, реализующую пузырьковую соритировку элементов массива. Проверить на разных массивах. Код отформатировать.
  2. Нарисовать блок-схему программы. Блок схему оформить с помощью сервиса draw.io.

Код отправить текстовым файлом, блок-схему в формате png.

Где почитать

  • Базовое форматирование кода
  • Пузырьковая сортировка и все-все-все
  • Сортировка прямым обменом (метод "пузырька")
  • Блок-схема

Практическая работа № 5. Двумерные массивы

Задачи:

  1. сумма всех элементов двумерного массива;
  2. сложение двух;
  3. умножение матрицы на число;
  4. умножение матриц;
  5. транспонирование матрицы;
  6. определитель матрицы 3х3;
  7. обратная матрица 3х3.

Где почитать

  • Массивы в C (часть 2): многомерные массивы
  • Операции над матрицами и их свойства
  • Матрицы и операции над ними

Практическая работа № 6. Прикладные задачи

Данные задачи связаны не только с программированием. В эту практическую добавлены элементы исследования задачи.

На сайте EasyFizika есть задачи из курса физики. Каждый для себя подбирает подходящую и работает с ней самостоятельно.

Этапы работы:

  1. выбрать задачу на сайте EasyFizika и обсудить выбранную задачу с преподавателем;
  2. запрограммировать решение на языке Си, отправить решение преподавателю;
  3. решить задачу на множестве входных данных (множество входных данных можно вписать вручную или генерировать в процессе работы программы);
  4. обработать результаты в программе LibreOffice Calc (построить графики).
